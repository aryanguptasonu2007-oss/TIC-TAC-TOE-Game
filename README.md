# 🎮 Tic-Tac-Toe Game

A fun and interactive **Tic-Tac-Toe** game built using **HTML, CSS, and JavaScript**.  
This project demonstrates core web development skills and provides a classic game experience directly in the browser.  
Play it live here: [Tic-Tac-Toe Website](https://aryanguptasonu2007-oss.github.io/TIC-TAC-TOE-Game/)

---

## 📖 Overview
Tic-Tac-Toe is a two-player strategy game played on a 3×3 grid. Players take turns marking cells with **X** or **O**.  
The goal is to align three of your marks horizontally, vertically, or diagonally. If all cells are filled without a winner, the game ends in a draw.

---

## ✨ Features
- **Two-player mode**: Play with a friend on the same device.
- **Win/draw detection**: The game automatically checks for winning combinations or a tie.
- **Reset button**: Restart the game instantly after each round.
- **Responsive design**: Works on desktop and mobile browsers.
- **Lightweight**: No frameworks—just pure HTML, CSS, and JavaScript.

---

## 🚀 How to Play
1. Open the game in your browser using the live link above.
2. Player **X** always starts first.
3. Players take turns clicking on empty cells to place their mark.
4. The game announces the winner or declares a draw when the grid is full.
5. Click **Reset** to start a new round.
